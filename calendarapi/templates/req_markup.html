Можна використовувати HTML теги для форматування тексту.

  • <p>Абзац тексту</p>

  • <center>текст по центру</center>
  • <strong>жирний</strong> <em>курсивний</em> <ins>підкреслений</ins> <del>закреслений</del>
  • <br>перехід на новий рядок(закривати його не потрібно)
  • Ненумерований список
      <ul>
    • <li>перший елемент</li>
    • <li>другий елемент</li>
      </ul>
  • Нумерований список
      <ol>
    1. <li>перший елемент</li>
    2. <li>другий елемент</li>
      </ol>
Теги для форматування можна комбінувати:

<p><strong>Перший абзац тексту жирним</strong> та <em>курсивом</em>. Перед списками обов'язково закриваємо тег абзацу.
<br>Новий рядок. Закриваємо тег абзацу</p>

    <ul>Заголовок ненумерованого списку(текст необов'язковий)
  • <li>перший <strong>елемент жирним</strong></li>
  • <li>другий <em>елемент курсивом</em></li>
    закриваємо тег списку </ul>

<br> (пропустимо рядок між cписком та абзацом)

<p><Другий абзац тексту. Текст <strong>обов'язково</strong> з обох боків має огортатися тегами абзацу(<p>) або тегами списку(</ul(або ol)</li>). Якщо вказати текст без тегів то форматування може бути некоректним.</p>
